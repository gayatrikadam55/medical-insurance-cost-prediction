# Medical Insurance Cost Prediction

## Problem Statement
The objective of this project is to predict medical insurance charges based on personal and health-related attributes using machine learning regression techniques.

## Real-World Relevance
Accurate prediction of medical insurance costs helps insurance companies design fair premiums and allows individuals to understand how lifestyle and demographic factors affect healthcare expenses.

## Dataset
Source: Kaggle  
Size: ~1338 records  
Target variable: charges  

## Type of ML Problem
Regression

## Expected Outcome
A regression model that can estimate medical insurance costs with reasonable accuracy based on input features.
